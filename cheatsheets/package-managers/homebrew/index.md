# Homebrew

## Related

- [Homebrew cheatsheet](https://devhints.io/homebrew) on DevHints.

## Common commands

### Install

```sh
$ brew install PACKAGE
$ brew install python
```

Specific version:

```sh
$ brew install PACKAGE@VERSION
$ # e.g.
$ brew install python@3.9
```

## Uninstall

```sh
$ brew remove PACKAGE
```

### Upgrade all packages

```sh
$ brew upgrade
```

### List installed packages

```sh
$ brew list
```

## Install Devtools

```sh
$ xcode-select --install
```
