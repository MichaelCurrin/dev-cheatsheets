## Configure

From [issue comment](https://github.com/pypa/pipenv/issues/2122#issuecomment-386207878).

```sh
echo 'export PATH="${HOME}/.local/bin:$PATH"' >> ~/.bashrc
```


## Install

Install with `pip`.

```sh
$ pip install pipenv
```

Or more precisely:

```sh
$ python3 -m pip install --user pipenv
```
