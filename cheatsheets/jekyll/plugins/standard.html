Setup

How to set up a plugin

  1. Add to your Gemfile.
  2. Run bundle install locally.
  3. Add the plugin to the config‚Äôs plugins section.
  4. Follow any plugin-specific set up like configuring and adding your layouts.

    • ‚ÄúA Jekyll plugin to generate an Atom (RSS-like) feed of your Jekyll posts.‚Äù_

SEO tag

  • Repo: jekyll-seo-tag
  • This adds metadata for search engines to use, including title, meta and a canonical tag. If you use this plugin, you don‚Äôt need to set title yourself or in a theme layout (as you‚Äôd duplicate the tag).
  • After installing, add to your head HTML tag such as in _layouts/default.html.

  • See the advanced usage in the docs for setting authors and image OG data.
