Arrays

Cheatsheets / JavaScript / General / Data structures / Arrays

See Array cheatsheet on ShortCutFoo website.

See map reduce filter page for how to apply those to an array.

View

const myArray = ['a', 'b', 'c', 'd']

Length

myArray.length
// 4

Get element by index

myArray[index]

e.g.

myArray[1]
// 'b'

myArray[4] // there is not 5th element
// undefined

Slice

See also splice method.

Get array of elements between a range. If you omit the second argument, it is implied as the lenght of the array (i.e. up to the end of the array).

myArray.slice(START_INDEX)

myArray.slice(START_INDEX, END_INDEX)

e.g.

Start from a given index and go to the end.

myArray.slice(1)
// [ 'b', 'c', 'd' ]

Start at an index (inclusive) and go up to an index (exclusive).

myArray.slice(0, 1)
// [ 'a' ]
myArray.slice(0, 2)
// [ 'a', 'b' ]

myArray.slice(1, 2)
// ['b']

myArray.slice(1, 3)
// [ 'b', 'c' ]

Using negative index:

myArray.slice(-1) // last
// [ 'd' ]

myArray.slice(-2) // last 2
// [ 'd', 'e' ]

myArray.slice(-3, -1) // 3rd last and 2nd last
//[ 'b', 'c' ]

Get last element as a scalar

Instead of getting as an array, we want to get as scalar (e.g. string or number).

Use hard bracket slicing

ARRAY[INDEX]

e.g.

myArray = ['a', 'b', 'c', 'd']
myArray[myArray.length - 1]
// 'd'

myArray = ['a']
myArray[myArray.length - 1]
// 'a'

Warning In Python, you simply use a negative index and hard brackets like my_list[-1]. But in JS, that doesn‚Äôt work.

myArray[-1]
// undefined

Use slice method

A slice is returned as an array. So to get the last element as a scalar (like a string), then use need to get the element from the array.

myArray.slice(-1)[0]
// 'd'

Update

Add, remove and update elements in an array.

Assign value

Basis syntax for setting a value:

> myArray[index] = value

See also Pop, Push, Shift and Unshift Array Methods in JavaScript

Modify end

Remove last element.

> myArray.pop();

Insert element at the end. i.e. append.

> myArray.push(obj)

Modify at the start

Remove from the beginning. Like Python‚Äôs list.pop(0). Returns new length.

> myArray.shift()

Insert at the start. Like Python‚Äôs list.insert(0, obj). Returns new length.

> myArray.shift(obj)

Change order

Reverse.

> myArray.reverse()

Sort. See Sort for more info, as this will not sort numbers properly.

> myArray.sort()

Join arrays

Concatenate arrays.

> myArrayA.concat(myArrayB)

Join elements using a separator e.g. ', '. This will return a single string.

> myArray.join(sep)

Splice an array

See also slice method that only reads elements from a range.

The splice reads a range too, but also deletes the items from the original array.

const oldArray = ['a', 'b', 'c', 'd']
// [ 'a', 'b', 'c', 'd' ]

const newArray = oldArray.splice(1, 2)
// [ 'b', 'c' ]

// Items have been removed.
oldArray
// [ 'a', 'd' ]

Iterate over an array

Use map to get values

const myArray = ['a', 'b', 'c']

myArray.map(value=> value.toUpperCase())
// [ 'A', 'B', 'C' ]

Get value and index

const myArray = ['a', 'b', 'c']

myArray.map((value, index) => [index, value.toUpperCase() ])
// [
//   [ 0, 'A' ],
//   [ 1, 'B' ],
//   [ 2, 'C' ]
// ]

const myArray = ['a', 'b', 'c']

for (const [index, value] of Object.entries(myArray)) {
  console.log(index, value.toUpperCase())
}
// 0 A
// 1 B
// 2 C

Contains

Check if array contains a value

const x = ['abc', 'def']

x.includes('abc')
// true

Get index of a value

const x = ['abc', 'def']

x.indexOf('abc')
// 0
x.indexOf('abcd')
// -1

Array and spread operator

Duplicate an array

const myArray = [ 'a', 'b', 'a']

[...myArray]
// [ 'a', 'b', 'a' ]

Array.from(myArray)
// [ 'a', 'b', 'a' ]

Convert iterable to an array

const mySet = new Set([ 'a', 'b', 'a'])

[...mySet]
// [ 'a', 'b' ]

// More verbose, older syntax, used internally by spread.
Array.from(mySet)
// [ 'a', 'b' ]

Treat iterable as an array

Process to follow if you need to convert a document selection to an array so you can use array methods on it like .map.

const divs = document.getElementsByTagName('div')
// Not available:
//   divs.map
//   divs.forEach

// Convert it to an array.
const divsArr = [...divs]

divsArr[0]
// <div class="position-relative js-header-wrapper ">...</div>

// Now use `.map`.
divsArr.map(i => i.innerText)
