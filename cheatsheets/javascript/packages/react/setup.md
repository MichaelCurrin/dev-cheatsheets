# Set up index


How to set up the index module of your React app. This is the entrypoing which loads your `App.jsx` file.

- `index.jsx`
    ```jsx
    ReactDOM.render(
    <React.StrictMode>
      <App />
    </React.StrictMode>,
    document.getElementById("root"),
    );
    ```
- `index.jsx`
    ```jsx
    ReactDOM.hydrate(
       <App />,
       document.getElementById('root')
    );
    ```
