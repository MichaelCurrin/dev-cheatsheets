Export

You must use this so that an object is available to import from another script.

export

Add an export line at the end of your script.

function foo () {}

class Bar () {}

// This is more modern and uses a cleaner hash.
export {
  foo,
  Bar
}
// Same as:
module.exports = {
  foo: foo,
  Bar: Bar
}

Or export in place:

export function foo () {}

export class Bar () {}
