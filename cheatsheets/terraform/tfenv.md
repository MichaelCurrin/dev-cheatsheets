# TFEnv

If you want to have multiple versions of Terraform installed, use this tool.

- [tfenv](https://github.com/tfutils/tfenv)


## Install TFEnv

```sh
$ brew install tfenv
```

## Install a version of Terraform

```sh
$ tfenv install 1.2.3
```


## Use a version of Terraform

```sh
$ terraform1.2.3
```
