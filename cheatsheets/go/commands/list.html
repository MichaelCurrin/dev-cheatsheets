Run 'go help list' for details.

From the CLI help docs:

The -m flag causes list to list modules instead of packages.

The default output is to print the module path and then information about the version and replacement if any.

For example, ‚Äògo list -m all‚Äô might print:

The -u flag adds information about available upgrades. When the latest version of a given module is newer than the current one, list -u sets the Module‚Äôs Update field to information about the newer module. The Module‚Äôs String method indicates an available upgrade by formatting the newer version in brackets after the current version. For example, ‚Äògo list -m -u all‚Äô might print:
