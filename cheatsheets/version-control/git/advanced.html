Get commit hash

Commit reference is required for both below.

Single

Using git rev-parse.

$ git rev-parse HEAD 
9641372537f8025591b8808c6f9194f211062394

$ git rev-parse --short HEAD 
9641372

Range

Using git rev-list.

$ g rev-list HEAD

Range of commits

Using git log.

From latest commit

$ git log

Show diff

$ git log -p

Compare branches or remote

$ git log main origin/main

$ git log main my-feature

Path

$ git log my-dir

$ git log my-file.txt

Show diff

Using git diff.

Only shows uncommit changes.

$ git diff

Show history.

$ git diff HEAD~

$ git diff HEAD~5..HEAD~2

$ git diff main my-feat
