## Email

- Markdown
	```markdown
	<foo@bar.com>
	```
- HTML
	```html
	<a href="mailto:foo@bar.com">foo@bar.com</a>
	```
- Result
	- <foo@bar.com>


## Phone number

- Markdown
	```markdown
	[+44 20 7123 4567](tel:+442071234567)
	```
- Rendered HTML
	```html
	<a href="tel:+442071234567">+44 20 7123 4567</a>
	```
- Result
	- [+44 20 7123 4567](tel:+442071234567)
