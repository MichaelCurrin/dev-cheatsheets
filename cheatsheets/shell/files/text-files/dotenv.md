# Dotenv

> A load a dotenv file.

Export the vars in `.env` file into your shell.

Simple

```sh
export $(< .env) | xargs)
```

Check if it exists first and ignore comments.

```sh
if [ -f .env ]; then
  export $(egrep -v '^#' .env | xargs)
fi
```
