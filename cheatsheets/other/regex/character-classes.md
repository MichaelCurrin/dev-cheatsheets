# Character classes

| Symbol        | Description                           |
| ------------- | ------------------------------------- |
| `\d `         | digit. Same as `[0-9]`                |
| `\D`          | not a digit                           |
| `\w`          | word character  - same as `[a-z]`     |
| `\W`          | not word character                    |
| `[a-z]`       | Lowercase alphabet                    |
| `[a-zA-Z]`    | Any letter of the alphabet, any case. |
| `[a-zA-Z0-9]` | Alphanumeric                          |
| `\s`          | whitespace                            |
| `\S`          | not whitespace                        |
| `.`           | any character, except a newline       |
| `\.`          | literal dot                           |
| `\n`          | Newline                               |
| `:alpha:`     | Letter                                |
| `:digit:`     | Digit                                 |
| `:space:`     | Space                                 |
