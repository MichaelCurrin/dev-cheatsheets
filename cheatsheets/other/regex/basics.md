# Basics

| Symbol | Description:                                                                                                |
| ------ | ----------------------------------------------------------------------------------------------------------- |
| `^`    | Start of a string                                                                                           |
| `$`    | End of a string                                                                                             |
| `.`    | Wildcard - matches any character, except a newline (`\n`).                                                  |
| `|`    | Matches a specific character or group of characters on either side (e.g. `a | b` corresponds to `a` or `b`) |
| `\`    | Escape a special character                                                                                  |
