## Examples

### Argparse

What the usage and help sections can appear - here using [Python](#python) `argparse`.

#### Positional arguments

```sh
python prog.py -h
```
```
usage: prog.py [-h] [--sum] N [N ...]

Process some integers.

positional arguments:
 N           an integer for the accumulator

optional arguments:
 -h, --help  show this help message and exit
```

Note that is help is included as an optional argument by default.

#### Optional arguments

Here is a flag with a value.

```sh
python myprogram.py --help
```
```
usage: myprogram.py [-h] [--foo FOO]

optional arguments:
 -h, --help  show this help message and exit
 --foo FOO   foo help
```

#### Groups

```
usage: PROG [--bar BAR] foo

group1:
  group1 description

  foo    foo help

group2:
  group2 description

  --bar BAR  bar help
  ```

#### Subcommands

```
usage:  [-h] {foo,bar} ...

optional arguments:
  -h, --help  show this help message and exit

subcommands:
  valid subcommands

  {foo,bar}   additional help
```

## Usage guidelines

Help on writing usage instructions, especially when writing by hand.


### List of items

Zero or more items:

```
command [ITEM [ITEM ...]]
```

One or more items:

```
command ITEM [ITEM ...]
```

Often a comma is used in these usage instructions but it can be misleading if a comma is not needed in the literal answer.
