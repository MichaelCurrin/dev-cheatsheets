# Align


## Text align

From [text-align](https://www.w3schools.com/cssref/pr_text_text-align.ASP) on W3 schools.

```css
el {
  text-align: center;
}
```

Values: `left|right|center|justify|initial|inherit`

```css
el {
  align-content: center;
}
```


## Align content

Use of `align-content` is for [Flexbox][].
