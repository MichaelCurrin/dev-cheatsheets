# User action

- `:focus` - using keyboard (or mouse)
- `:hover` - hover with mouse)
- `:checked` for certain for elements
- `:visited` for links


## Examples

### Button

```css
/* Any button over which the user's pointer is hovering */
button:hover {
  color: blue;
}
```

### Links

```css
a:link,
a:visited {
    color: rebeccapurple;
    font-weight: bold;
}

a:hover {
    color:hotpink;
}  
```
