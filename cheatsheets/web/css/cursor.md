# Cursor

Set the appearance of the mouse pointer.


## Syntax

```css
selector {
  cursor: VALUE;
}
```


## Values

Some recommended values:

- `pointer` for a link. This is already built-in for `a` tags but you might want to add this for buttons.
- `help`
- `default`
- `wait` or `wait`
- `not-allowed` or `no-drop`
