Check list

Check all values are truthy i.e. not of one of False, 0, [], {}.

values = [True, True, True]
all(values)
# True

values = [True, False, True]
all(values)
# False

values = [1, 2, 3]
all(values)
# True

values = [0, 1, 2, 3]
all(values)
# False

Match

Added in Python 3.10.

See post.

match subject:
    case <pattern_1>:
        <action_1>
    case <pattern_2>:
        <action_2>
    case <pattern_3>:
        <action_3>
    case _:
        <action_wildcard>

A simple example of literal matching.

def http_error(status):
    match status:
        case 400:
            return "Bad request"
        case 404:
            return "Not found"
        case 418:
            return "I'm a teapot"
        case _:
            return "Something's wrong with the internet"

case 401 | 403 | 404:
    return "Not allowed"
