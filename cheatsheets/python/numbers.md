# Numbers

## Float

```python
1.2

0.2

.2
```


## Integer

```python
123
```

Use an underscore for readibility

```python
10_000_000_000
```

## Division

Integer division always retursn a float.

```python
>>> 2/2
1.0
```

```python
>>> 123 / 2
61.5
```

Floor divison

```python
>>> 123 // 2
61
```
